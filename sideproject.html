Fino Tratto

Descrizione

Fino Tratto è un ristorante di isolalta di Vigasio con cucina brasiliana, io mi sono occupato della realizzazione del sito utilizzando HTML, css, js e framework di frontend come bootstrap e jkelly. il codice è visualizzbile su github con cui ho gestito in toto il versionamento. mi sono occupato anche dell'hosting, ho utilizzato github pages per l'hosting e squarespace per il dominio, utilizzando due servizi diversi per dominio ed hosting ho anche docuto configurare il dns di rete.
