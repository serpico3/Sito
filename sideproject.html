TaskFlow App

Applicazione per la gestione delle attività

Descrizione

TaskFlow è un'applicazione web completa per la gestione delle attività personali e di team. Offre un'interfaccia intuitiva con funzionalità avanzate come:

  • Drag & drop per riorganizzare le attività
  • Sistema di tag e categorie personalizzabili
  • Collaborazione in tempo reale tra utenti
  • Notifiche e promemoria
  • Statistiche e report di produttività

Tecnologie utilizzate

React Redux Firebase Node.js

EcoTracker

Monitoraggio impronta ecologica

Descrizione

EcoTracker è un'applicazione mobile che aiuta gli utenti a monitorare e ridurre il loro impatto ambientale quotidiano. Le principali funzionalità includono:

  • Tracciamento delle emissioni di CO2
  • Calcolo dell'impronta ecologica
  • Suggerimenti personalizzati per ridurre l'impatto
  • Community per condividere risultati e obiettivi
  • Integrazione con dispositivi IoT smart home

Tecnologie utilizzate

Vue.js Express MongoDB PWA

RecipeAI

Generatore di ricette con AI

Descrizione

RecipeAI utilizza l'intelligenza artificiale per generare ricette personalizzate basate sugli ingredienti disponibili e sulle preferenze dietetiche. Caratteristiche principali:

  • Generazione di ricette in tempo reale
  • Adattamento a restrizioni dietetiche (vegano, senza glutine, ecc.)
  • Sistema di valutazione e feedback
  • Lista della spesa automatica
  • Integrazione con assistenti vocali

Tecnologie utilizzate

Python Flask OpenAI API PostgreSQL
